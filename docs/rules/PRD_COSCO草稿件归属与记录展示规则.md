## 需求背景

当前开发实现存在将 COSCO 草稿件归入「预览件中」的情况，该处理不符合业务逻辑。

COSCO 船司存在草稿件结果物，草稿件应作为独立结果物展示在「草稿件」tab 中，不应被归并到「预览件记录」或「预览件中」状态里。草稿件当前仅支持生成一次，页面需展示该草稿件对应的最新一次提交记录、回执时间、文件信息、比对状态，以及查看不一致内容能力。

同时，全局所有「提交截图」和「查看不一致内容」均需要统一改为遮罩类弹窗，避免跳转或零散展示导致用户上下文丢失。

## 适配范围

- 适配船司：COSCO
- 适配客户：全客户
- 外部系统：无

## 功能及界面设计

### 1. COSCO 草稿件归属规则

- COSCO 草稿件必须展示在「草稿件」tab 中。
- COSCO 草稿件不得展示在「预览件记录」或「预览件中」状态中。
- 「预览件记录」仅展示预览件相关记录，不包含 COSCO 草稿件。
- 若 COSCO 已生成草稿件，用户进入制单详情后，可在「草稿件」tab 查看草稿件结果物。

### 2. 草稿件生成与展示规则

- COSCO 草稿件当前仅支持生成一次。
- 若已存在草稿件，再次触发生成草稿件时，不重复生成新的草稿件结果物。
- 草稿件 tab 展示当前草稿件的最新一次提交相关信息。
- 草稿件 tab 展示字段：

| 字段 | 说明 |
|---|---|
| 最新一次提交记录 | 展示该草稿件对应的最新提交记录入口 |
| 回执时间 | 展示船司回执时间 |
| 文件信息 | 展示草稿件文件名称、下载、联系客户等操作 |
| 比对状态 | 展示比对一致 / 比对不一致状态 |
| 查看不一致内容 | 比对不一致时展示，点击后打开遮罩弹窗 |

### 3. 提交记录入口

- 草稿件 tab 和 VC件 tab 中保留「提交记录」入口。
- 入口放在对应 tab 内容区域内，不放在详情主页外层。
- 点击「提交记录」后展示提交记录弹框。
- 提交记录弹框展示最新一次和历史提交记录，按提交次数倒序。

### 4. 提交截图遮罩弹窗

- 草稿件 tab 和 VC件 tab 中的「提交截图」入口统一改为遮罩类弹窗。
- 预览件记录中的「查看预览件」不打开提交截图遮罩弹窗，保留原预览件查看逻辑。
- 点击入口后，在当前页面上方打开遮罩弹窗，不跳转页面。
- 弹窗内容展示当次提交船司时的页面快照。
- 快照为只读，不支持编辑。
- 弹窗内容参考「确认提单信息」样式展示，包括：
  - 船司、账号、提单号
  - 发货人、收货人、通知人
  - 文件信息、付款信息
  - 箱信息
  - 货物信息
  - 通知信息
- 点击右上角关闭、遮罩、返回或确认按钮，关闭弹窗。

### 5. 查看不一致内容遮罩弹窗

- 全局所有「查看不一致内容 / 查看不一致字段」入口统一改为遮罩类弹窗。
- 预览件记录区域中的不一致字段弹框默认隐藏。
- 预览件记录中点击「查看不一致字段」后，打开不一致内容遮罩弹窗。
- 预览件记录中如存在「查看预览件」入口，仍按预览件查看逻辑处理，不打开提交截图遮罩弹窗。
- 点击入口后，在当前页面上方打开遮罩弹窗，不跳转页面。
- 弹窗表格字段：

| 字段 | 说明 |
|---|---|
| 字段 | 不一致字段名称 |
| 系统值 | 系统侧字段值 |
| 三方值 | 船司 / 三方返回字段值 |
| 原因 | 不一致原因 |

- 示例字段：
  - 总件数：系统值 6，三方值为空，原因「值不匹配」
  - 总体积：系统值 266.5，三方值 263.5，原因「值不匹配」
  - 总重量：系统值 39288.01，三方值 39286，原因「值不匹配」
- 点击右上角关闭、遮罩或确定按钮，关闭弹窗。

## 数据与状态规则

- COSCO 草稿件与预览件是不同结果物，不得共用预览件记录展示区。
- 草稿件仅生成一次，以提单号维度判断是否已存在草稿件。
- 草稿件 tab 展示最新一次提交记录，不展示多份草稿件结果物。
- 比对状态为「比对不一致」时，展示「查看不一致内容」。
- 提交截图以提交动作发生时的页面状态为准，后续字段变更不影响历史截图。

## 验收标准

1. Given 船司为 COSCO 且存在草稿件，When 用户进入制单详情并点击「草稿件」tab，Then 页面展示 COSCO 草稿件结果物。
2. Given 船司为 COSCO 且存在草稿件，When 用户查看「预览件记录」或「预览件中」状态，Then 不展示该草稿件结果物。
3. Given COSCO 已生成草稿件，When 再次触发生成草稿件，Then 系统不重复生成第二份草稿件结果物。
4. Given 草稿件存在提交记录，When 用户进入草稿件 tab，Then 页面展示最新一次提交记录入口、回执时间、文件信息和比对状态。
5. Given 草稿件比对状态为不一致，When 用户点击「查看不一致内容」，Then 系统打开遮罩弹窗展示字段、系统值、三方值和原因。
6. Given 用户在提交记录中点击「查看截图」，When 操作触发，Then 系统打开遮罩弹窗展示当次提交截图。
7. Given 任意遮罩弹窗已打开，When 用户点击关闭、遮罩、返回、确定或按 Esc，Then 弹窗关闭并返回当前详情页上下文。
8. Given 用户位于预览件记录 tab，When 页面初始展示，Then 不一致字段弹框默认隐藏。
9. Given 用户位于预览件记录 tab，When 用户点击「查看不一致字段」，Then 系统打开不一致内容遮罩弹窗。
10. Given 用户位于预览件记录 tab，When 用户点击「查看预览件」，Then 不打开提交截图遮罩弹窗。
11. Given 用户点击任意「查看不一致内容」或「查看不一致字段」，When 操作触发，Then 系统打开不一致内容遮罩弹窗。
